<h2> C calculator written with the ncurses.h library </h2>

This is a basic calculator that handles basic arimethic operations ( addition, subtraction, multiplication and division)

<h3> What do I need to run this program?</h3>
You need:

-  Linux as OS
-  ncurses.h header files and libraries (in current distros, these are included by default)
-  A C compiler

<h3>How do I compile it?</h3>

If you're using GCC, open the terminal and type this:

`gcc test2.c -o test2 -lncursesw`
